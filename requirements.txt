The following Python libraries were used in the writing of the programs code and are required to successfully run the notebooks:

* Matplotlib.pyplot - Used for the manipulation of elements and the creation of certain plots graphs, plots and charts within the scikit-learn.ipynb and scipy-stats.ipynb notebooks. Can be be found at: https://matplotlib.org/tutorials/introductory/pyplot.html.

* Scipy - Python library used for scientific computing and technical computing within the scikit-learn.ipynb and scipy-stats.ipynb notebooks. Can be be found at: https://scipy.org/.

* Seaborn - Used for the creation and manipulation of plots in the scikit-learn.ipynb and scipy-stats.ipynb notebooks. Can be be found at: https://seaborn.pydata.org/.

* Statsmodels - A python package to explore data, estimate statistical models and perform statistical tests in the scikit-learn.ipynb and scipy-stats.ipynb notebooks. Can be be found at:https://www.statsmodels.org/stable/index.html/.
